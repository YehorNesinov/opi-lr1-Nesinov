## Встановлення та запуск

Для запуску проєкту локально виконайте наступні команди в
терміналі:

```bash
git clone [https://github.com/YehorNesinov/opi-lr1-Nesinov.git](https://github.com/YehorNesinov/opi-lr1-Nesinov.git)
cd opi-lr1-Nesinov
```
